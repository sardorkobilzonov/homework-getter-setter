Map/Set

Map

Description
Map ob'ektlari kalit-qiymat juftliklari to'plamidir. Mapdagi kalit faqat bir marta paydo bo'lishi mumkin; Map kolleksiyasida noyobdir. Map obyekti kalit-qiymat juftliklari orqali takrorlanadi — for...of tsikli har bir iteratsiya uchun 2 aʼzoli [kalit, qiymat] array qaytaradi.
const wrongMap = new Map();
wrongMap["bla"] = "blaa";
wrongMap["bla2"] = "blaaa2";
console.log(wrongMap);
// Map { bla: 'blaa', bla2: 'blaaa2' }

Set

Description
Set ob'ektlari qiymatlar to'plamidir. To'plamdagi qiymat faqat bir marta bo'lishi mumkin; u to'plam kolleksiyasida noyobdir. Siz qo'shish tartibida to'plam elementlarini takrorlashingiz mumkin.
const a = new Set([1, 2, 3]);
const b = new Map([
[1, "one"],
[2, "two"],
[4, "four"],
]);
console.log(a.union(b));
// Set(4) {1, 2, 3, 4}
